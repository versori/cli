# Versori Skills

A collection of AI agent skills for working with the [Versori](https://versori.com) platform. Skills are an open standard for packaging AI agent knowledge and workflows, usable across AI coding tools such as Claude Code and Cursor.

## Available Skills

| Skill | Description |
|-------|-------------|
| [coding-versori-sdk](./coding-versori-sdk/README.md) | Helps write expert-level data integration code using the versori-run SDK — ETL processes, API integrations, data pipelines, webhooks, and more. |
